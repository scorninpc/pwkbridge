# PHP Webkit Bridge
Create the bridge between PHP and Webkit to create desktop applications using HTML tecnologi and PHP local server

# QTWebKit
``
$ apt install qt5-default libqt5webkit5-dev
``

# Compile 
```
$ QT_SELECT=qt5 qmake
$ make
```
